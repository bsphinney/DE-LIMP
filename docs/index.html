DE-LIMP

An interactive R Shiny application for differential expression analysis of DIA-NN proteomics data, powered by Limpa and Limma.

View on GitHub Download Latest Release

Key Features

Interactive Volcano Plots

Fully interactive Plotly volcano plots. Click or box-select proteins to highlight them across all views.

AI-Powered Analysis

Chat with your data using Google Gemini. Ask biological questions and get context-aware answers with automatic protein highlighting.

QC Trend Monitoring

Track precursor and protein counts across run order to identify batch effects and quality issues early.

Deep-Dive Grid View

Heatmap-style expression grid with color-coded values, UniProt links, and click-to-plot violin charts.

Reproducibility Log

Every analysis step is logged as executable R code with timestamps. Download a complete, runnable script of your session.

Save & Load Sessions

Save your entire analysis state as an RDS file and reload it later to continue exactly where you left off.

Getting Started

  1. Install R 4.5+

    DE-LIMP requires R version 4.5 or newer for Bioconductor 3.22+ compatibility. Download from cloud.r-project.org.

  2. Clone the repository

    git clone https://github.com/bsphinney/DE-LIMP.git
cd DE-LIMP

  3. Run the app

    The app will automatically install any missing packages on first launch.

    Rscript -e "shiny::runApp('DE-LIMP.R', port = 3838, launch.browser = TRUE)"

  4. Upload your DIA-NN data

    Use the sidebar to upload a DIA-NN parquet or TSV report, assign experimental groups, and run the analysis pipeline.

Analysis Pipeline

DE-LIMP uses the LIMPA package to run a well-established proteomics analysis workflow:

  1. Data Import

    Load DIA-NN report files (parquet or TSV) with readDIANN().

  2. Normalization

    Apply Data Point Correspondence (DPC-CN) normalization via dpcCN() to correct for systematic biases.

  3. Protein Quantification

    Roll up peptide-level data to protein-level using maxLFQ quantification with dpcQuant().

  4. Differential Expression

    Fit linear models with dpcDE(), apply contrasts, and compute empirical Bayes moderated statistics with eBayes(). FDR correction via Benjamini-Hochberg.
