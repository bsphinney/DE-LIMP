Methods Summary
R Code Log

Proteomics data analysis. DIA-NN output was processed using the limpa pipeline in DE-LIMP v3.0. Precursor-level data was normalized using DIA-NN's RT-dependent method, aggregated to protein level via Top3 (dpcQuant), and differential expression performed with limma empirical Bayes (dpcDE). FDR correction: Benjamini-Hochberg at α = 0.05, minimum |log2FC| = 1.0.

Gene set enrichment. GSEA was performed using clusterProfiler with GO Biological Process terms. Genes ranked by signed -log10(p) × sign(logFC)…
